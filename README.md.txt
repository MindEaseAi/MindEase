# Mind Ease

A fully working calming web app with aurora visuals, glowing effects, calming activities, and audio. 

## How to deploy on GitHub Pages:

1. Create a new GitHub repository.
2. Upload all files from this folder (`index.html` and `README.md`).
3. Go to **Settings → Pages**.
4. Under **Source**, choose **main branch → / (root)**.
5. Click **Save**.
